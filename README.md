# Tony_Jurnal — Дневник поездки на скутере

Учебный проект из книги **«Head First HTML and CSS»** — дневник путешествия
Тони на скутере по США. Статическая HTML-страница на русском языке.

## Файлы

- `index.html` — Дневник: записи, таблица городов, цитаты
- `jurnal.css` — Стили (кастомный шрифт Emblema One, оранжевая схема)
- `fonts/` — Шрифт Emblema One (TTF + WOFF)
- `images/` — Фото и иконки

## Исправленные баги

- Добавлен атрибут `lang="ru"` к `<html>`
- Добавлен `<meta name="viewport">` для мобильных устройств
- Добавлены атрибуты `alt` ко всем изображениям
